Privacy Policy — 3D Viewer

Last updated: July 9, 2026

3D Viewer ("the app") is a free, client-side web application for viewing 3D model files (STL, OBJ, GLB, GLTF). This policy explains what data the app accesses and how it is used.

What the app is

3D Viewer runs entirely in your browser. There is no backend server and no database — the app does not store, log, or transmit your files or personal data to us or to any third party.

Google Drive access

If you choose to open a file from Google Drive, the app requests the drive.file OAuth scope. This scope only grants access to the specific file you explicitly select — either through the Google Picker file-selection dialog, or by using Drive's "Open with" menu on a file. The app cannot see, list, or access any other files in your Google Drive.

The selected file's contents are downloaded directly from Google's servers into your browser to render the 3D model. The file is never uploaded to, or stored on, any server operated by us.

Local files

Files you open using the "Open File from Computer" button never leave your device — they are read locally by your browser and are not transmitted anywhere.

Data we collect

We do not collect, store, or share any personal data, analytics, or usage tracking. The app does not use cookies.

Google API Services User Data Policy

3D Viewer's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
